# Font Files

This directory should contain the following font files:
- Inter-Regular.ttf
- Inter-Bold.ttf

These files are used for PDF rendering. You can download them from:
https://github.com/rsms/inter/tree/master/docs/font-files

Or use any other TTF fonts as needed.

For this implementation, we're using Inter as it's a clean, modern font that works well in resumes.
